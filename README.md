# zkMe

[zkMe](https://github.com/zTgx/zkMe) is a toy. 
 
## Supported build tools versions
g++ 8.4  +   
cmake 3.17 +  
rust-toolchain 1.52.0

## Building
```c++
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Debug ..
make
```

## Run
```c++
./zkme -groth16(plonk)
```
